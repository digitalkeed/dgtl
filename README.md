# Leart Raka — Portfolio

Standalone portfolio site for [leartraka.vercel.app](https://leartraka.vercel.app), inspired by [Chris Shu](https://chrisshu.com/) and [Agon Avdimetaj](https://agonavdimetaj.com/).

## Run locally

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

1. Push this repo to GitHub.
2. In [Vercel](https://vercel.com): **Add New Project** → import this repo.
3. Deploy. Add domain **leartraka.vercel.app** in Project → Settings → Domains.

## Edit content

Update `content/portfolio.ts`: name, tagline, about, ventures, email, and links (LinkedIn, Email, optional social).
